Personal Book Review App
About

This is my first personal Book Review App, designed to help me organize and track my thoughts on the books I read. It lets me record reviews, rate books, and keep a personal reading history.

Goals

Keep a simple, personal system to track book reviews.

Practice building an app from scratch as my first project.

Explore AI-assisted coding to help structure and solve challenges during development.
